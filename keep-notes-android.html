Keep Notes

Version 1.5

Overview

Keep Notes is an app built to store your notes, thoughts and ideas. It hopes to provide a clean experience while also adding useful functionalities that you'll use.

Changenotes

  • The layout of the recyclerview has been changed to be a staggered grid layout.
  • The note preview now shows a maximum of 3 lines.
  • A new search function has been added to find notes with titles, dates, times and details that match a search query.
  • Also fixed dates to be in an American format.
  • A trash menu has been added to restore deleted notes.
  • Added an animation to the main screen.

Keep Notes

Version 0.5

Introduction

This entry introduces Keep Notes for Android! After more than a year, Keep Notes is finally being developed for Android. With the help of online tutorials and videos, I've been able to create a fully functional notes app for Android using Java and Android Studio. The UI is in need of a major retouch, I'm aware, but I'm taking small steps. Android development is much more rigorous and demanding. I will work hard to build this app to be the best that it can be. In these journal entries, I'm hoping to show a bit more of the process, such as any testing or prototyping I did before releasing the final product.
